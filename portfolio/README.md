# Aryan Portfolio

Static portfolio website starter.

## Open locally
- Open `index.html` in browser or run a local server.

## Customize
- Update `index.html` text, links, and project cards.
- Edit `styles.css` colors/spacing.
- Add project detail pages or modal components.

## Publish
- Push to GitHub and use GitHub Pages.
- Or deploy to Netlify/Vercel/Cloudflare Pages.
